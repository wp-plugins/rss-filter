== Description ==

RSS Filter plugin .You have option to customization your WordPress RSS feed items.

= Features =
1. customization your blog rss.
2. sticky post show in your rss (feed).
3. exclude any category in your WordPress rss (feed).
4. exclude any post in your WordPress rss (feed).

More Detail : http://socialcms.wordpress.com/


== Installation ==


1. Upload the **rss-filter** folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go **RSS Filter** in admin menu and customization your WordPress RSS feed.
4. See your WordPress RSS feed.

== Frequently Asked Questions ==

* **Only Sticky Post show in rss feed?**

 Show Only Sticky Post check box set to checked .if no sticky post found that it work default like

* **How to Exclude Certain Categories From RSS Feed?**

 In Exclude Category text box enter category ids (comma separated)

* **How to Exclude Certain Post From RSS Feed?**

 In Exclude Post text box enter post ids (comma separated)
